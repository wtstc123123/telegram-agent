# Infrastructure notes

This starter keeps infrastructure intentionally light.

## Included
- Dockerfile for the API service
- docker-compose.yml for local development with Redis

## Suggested next additions
- Reverse proxy or ingress (Nginx, Caddy, ALB, Cloud Run ingress)
- Managed secret store
- Observability (OpenTelemetry, Grafana, CloudWatch)
- Mermaid CLI renderer container if you want PNG/SVG output
- CI/CD pipeline
